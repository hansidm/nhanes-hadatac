Getting started

Dependencies

The NHANES-KG is deployed using HADatAc, a scientific knowledge graph framework. In its turn, HADatAc is more easily deployed using Docker. While you are welcome to build the NHANES-KG from scratch using all the provided data and metadata, luckily, we have pre-built Docker images with NHANES-KG for you. Therefore, you’ll only need the following to begin:

  1. Docker Engine
  2. Docker Compose
  3. Git - you most likely already have this one.

Quick start: Run NHANES-HADatAc Docker containers

  1. Clone our Github repo somewhere in your machine 
      $ git clone https://github.com/tetherless-world/nhanes-hadatac.git
  2. Navigate to the docker compose file path 
      $ cd nhanes-hadatac/docker/
  3. Start the NHANES-KG services using docker compose up 
      $ docker-compose up -d
  4. Point your web browser to http://localhost:9000 to confirm everything is running as expected. You should see HADatAc’s initial dashboard with some numbers about the NHANES-KG.

In most Linux systems localhost will be used. However, in MacOS or Windows hosts you will need to check your environment as the Docker containers might be running virtualized. In these cases, it may be required to know the IP for the virtual machine.
